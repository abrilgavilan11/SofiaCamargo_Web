Este proyecto es la página web oficial de **Sofía Isis Camargo**, Concejal de la ciudad de Plottier por el bloque Evolución. 

El sitio está diseñado para funcionar como un puente directo entre la concejal y los vecinos, brindando un espacio digital moderno para visibilizar su trabajo legislativo, sus propuestas sociales y fomentar la participación ciudadana en la comunidad.

## 📌 ¿Qué vas a encontrar en esta web?

La plataforma está estructurada en secciones clave para facilitar la navegación y el acceso a la información:

* **Conoceme:** Una sección biográfica y de contacto directo, acompañada de una galería interactiva con imágenes de Sofía trabajando en el concejo y en los barrios.
* **Proyectos en Marcha:** Espacio dedicado a las iniciativas activas, como el programa sustentable **"Eco Canje"** y el **"Taller de Bicicletas"**. Cuenta con modales interactivos que permiten deslizar y leer los flyers informativos de cada propuesta.
* **Participá (Sumate a la Acción):** Un área diseñada para la comunidad, que integra un formulario interactivo para que los vecinos de Plottier puedan inscribirse y colaborar en las próximas jornadas de forestación y mejora de espacios verdes.
* **Accesos Institucionales:** Enlaces directos a las redes sociales oficiales y a la web del Concejo Deliberante de Plottier.

## ✨ Características Técnicas Destacadas

* **Arquitectura Modular en React:** El código está dividido en componentes reutilizables (Navbar, Hero, Carruseles, Modales), garantizando un rendimiento óptimo y facilidad de mantenimiento.
* **Estilos Escopados (CSS Modules):** Implementación de `.module.css` para mantener un diseño limpio, coherente y libre de conflictos visuales.
* **Diseño Responsivo:** Adaptación 100% fluida a dispositivos móviles, tablets y monitores de escritorio.
* **Interactividad UI/UX:** * Menú hamburguesa dinámico para navegación móvil.
  * Carruseles táctiles (`scroll-snap`) para la galería de imágenes y proyectos.
  * Modales con animaciones fluidas para expandir información sin recargar la página.

## 🛠️ Tecnologías Utilizadas

* **Frontend:** React 18
* **Herramienta de Construcción:** Vite
* **Estilos:** CSS3 puro con CSS Modules y variables globales (`:root`).

## 🚀 Instalación y Uso Local

Si querés clonar este proyecto y correrlo en tu máquina local, seguí estos pasos:

1. **Clonar el repositorio:**
   ```bash
   git clone https://github.com/abrilgavilan11/SofiaCamargo_Web.git
   ```

2. **Navegar al directorio:**
    ```bash
    cd web-sofia-camargo
    ```

3. **Instalar las dependencias:**
    ```bash
    npm install
    ```

4. Levantar el servidor de desarrollo:
    ```bash
    npm run dev
    ```
